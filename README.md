# nextjs-prisma-graphql-sample

## 概要

Next.js x Prisma x GraphQL を利用したサンプルアプリケーション

## 使用技術

- フロントエンド

  - Next.js
  - NextAuth.js (認証)

- ORM

  - Prisma

- クエリ言語

  - GraphQL

- その他
  - GraphQL Code Generator（型情報の自動生成に利用）
